# 技术选型评分器

帮助团队进行客观、可追溯的技术选型决策。

## 特点

- **多维度评分** - 从性能、成熟度、学习成本等多角度评估
- **权重模型** - 根据业务场景（MVP/核心系统/内部工具）动态调整权重
- **AI 辅助** - 自动收集客观数据并给出评分依据
- **风险推演** - 识别潜在风险并制定应对策略

## 使用方法

在 Code CLI 中调用：

```
tech-evaluator
```

## 工作流程

1. **理解选型场景** - 确认选型目标、业务场景、候选方案、约束条件
2. **设计评分维度** - 选择通用维度或自定义专属维度
3. **设计权重模型** - 根据场景分配各维度权重
4. **AI 辅助评分** - 收集数据，为每个方案打分
5. **风险推演** - 分析技术风险、生态风险、团队风险
6. **最终决策** - 输出完整的选型决策报告

## 适用场景

- 框架选型（前端/后端）
- 数据库选型
- 中间件选型
- 云服务选型
- 技术方案比较
